Formatting

This is the section for showcasing different ways to format text.

For instance, you can emphasize text, make text strong, or even make strongly emphasized text!

You can also make lists:

  1. First, you need to use the <ol> or <ul> tags to open the list.
  2. Then, you can use the <li> tag to add list items (don't forget to close them)!
  3. The list items will be automatically formatted differently depending on which type of list you are using.
  4. Once you're done adding items to the list, then you need to close the list using the appropriate closing tag.

And that's it! Lists can be useful for many things:

  • You can use them to list the steps for how to do something.
  • You can use them to list multiple items from a category.
  • You can even use them to list ways which you can use something!

All in all, lists are very useful, and can come in handy when making your website!
